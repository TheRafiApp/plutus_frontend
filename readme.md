<a name="installation"></a>
## Installation

1. Clone the repo 

    ```
  git clone git@github.com:TheRafiApp/plutus.git
    ```
2. Install NPM dependencies

    ```
  npm install
    ```
3. Install Bower dependencies

    ```
  bower install
    ```

<a name="server"></a>
## Server

1. Make sure Mongo is running

    ```
  mongod
    ``` 

<a name="build"></a>
## Build

1. Intiate gulp js-build task

    ```
  gulp js-build
    ```

<a name="debug"></a>
## Debug
